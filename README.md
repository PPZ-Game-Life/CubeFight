# CubeFight - 方块大乱斗

3D解谜合成 H5 小游戏，基于 Three.js 开发。

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 当前进度 (2026-03-14)

### ✅ 已完成核心功能

1. **轨道控制器**
   - 水平360度环绕
   - 俯仰角限制15°-80°
   - 地平线锁定（禁止翻滚）
   - 阻尼惯性效果

2. **核心玩法**
   - 3x3x3方块阵列渲染
   - 红黄蓝三色 + 9级等级系统
   - 蓝块合成（同级蓝+蓝 → 升级）
   - 蓝块吞噬（蓝吃红/黄，等级判定）

3. **智能高亮系统**
   - 选中蓝块后自动高亮可操作目标（绿色边框）
   - 不可操作方块自动灰显（透明度0.3）
   - 防呆设计，降低玩家心智负担

4. **Combo系统**
   - 3秒内连续操作触发Combo
   - 积分乘数（x2, x3, x4...）
   - 超时自动重置

5. **剖面切割**
   - X/Y/Z轴切面查看
   - 非当前层透明化 + 禁用碰撞
   - 解决3D遮挡痛点

## 操作说明

- **拖拽旋转**: 鼠标/触摸拖拽旋转视角
- **点击选中**: 点击蓝色方块选中
- **执行操作**: 选中后点击高亮的目标方块
- **切面查看**: 点击X/Y/Z切面按钮逐层查看
- **重置视角**: 点击"重置视角"按钮

## 技术栈

- **Three.js** - 3D渲染引擎
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **GSAP** - 动画库

## 项目结构

```
src/
├── core/
│   ├── Config.ts          # 游戏配置
│   ├── Game.ts            # 游戏主控制器
│   ├── SceneManager.ts    # Three.js场景管理
│   └── InputManager.ts    # 轨道控制器
├── entities/
│   ├── Cube.ts            # 方块实体
│   └── Grid3D.ts          # 3D网格数据结构
├── systems/
│   └── SliceSystem.ts     # 剖面切割系统
└── main.ts                # 入口
```
