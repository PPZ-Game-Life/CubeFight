# CubeFight - 方块大乱斗

## 项目概述
基于Three.js开发的3D解谜合成H5小游戏,目标平台为CrazyGames。

## 操作说明

### 基础操作
1. **旋转视角**: 鼠标拖拽空白处
2. **选中方块**: 点击蓝色方块
3. **执行操作**: 选中后点击高亮的目标方块
4. **取消选中**: 再次点击已选中的方块

### 切面视图
- **顶视图(Y)**: 查看Y轴切面
- **侧视图(X)**: 查看X轴切面
- **前视图(Z)**: 查看Z轴切面
- **重置视角**: 恢复默认视角

### 游戏规则
1. **蓝块合成**: 两个相邻的同级蓝块可以合成为更高级的蓝块
2. **蓝块吞噬**: 蓝块可以吃掉等级≤自己的红块或黄块
3. **黄块合成**: 两个相邻的同级黄块可以合成
4. **金币获取**: 蓝块吃掉黄块时掉落金币
5. **Combo系统**: 3秒内连续操作可以叠加积分乘数

## 项目结构
```
src/
├── core/
│   ├── Game.ts              # 游戏主控制器
│   ├── SceneManager.ts      # Three.js场景管理
│   ├── InputManager.ts      # 输入事件处理
│   └── Config.ts            # 全局配置
├── entities/
│   ├── Cube.ts              # 方块实体类
│   └── Grid3D.ts            # 3D阵列数据结构
├── systems/
│   └── SliceSystem.ts       # 剖面切割系统
└── main.ts                  # 入口文件
```
