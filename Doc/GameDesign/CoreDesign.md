# CubeFight - 核心设计文档 (GDD)

## 一、 游戏概述
- **游戏名称**: CubeFight (方块大乱斗)
- **游戏类型**: 3D解谜合成 / 休闲街机 H5 小游戏
- **目标平台**: CrazyGames
- **核心体验**: 空间预判的策略感 + 吞噬合成的数值爽感 + 纯粹的冲榜竞技
- **变现模式**: 免费游玩 + 激励视频广告 (无IAP)
- **产品策略**: 聚焦“纯无尽模式”，通过空间规模（2x2 到 5x5）的解锁提供长线追求，确保首发版本开发可控且极具耐玩度。

## 二、 核心玩法机制

### 1. 基本规则 (红黄蓝三色相克)
游戏在 3D 空间（立方体阵列）中进行，方块分为三种颜色，每种颜色分为 1~9 级。

- **蓝色方块 (战士 - 玩家控制端)**
  - **合成**: 两个相邻（水平或垂直，不可斜向）的同级蓝块点击可合成，生成一个 `原等级 + 1` 的新蓝块。
  - **攻击/吞噬**: 点击蓝块，再点击相邻的、**等级 ≤ 蓝块等级** 的红块或黄块，蓝块会将其吃掉并占据其位置。

- **红色方块 (怪物 - 障碍与挑战)**
  - **特性**: 不可被玩家主动合成。随着玩家操作腾出空位，系统会不断刷新红块填补。
  - **胜负判定**: 红色方块作为主要的障碍物，当整个 3D 棋盘被填满，玩家无法进行任何合成或吞噬操作时，判定为 **Game Over**。

- **黄色方块 (财宝 - 高风险高收益目标)**
  - **合成**: 与蓝块一样，两个相邻同级黄块可合成升级。
  - **收益**: 被蓝块吃掉时，根据黄块等级提供显著高于红块的积分收益。
  - **金黄色方块**: 黄色体系中的稀有高价值变体。被蓝块吃掉时，除高额积分外，还会掉落局内一次性道具。这是游戏核心的“贪心”策略点。

### 2. 核心操作与 3D 视角
- **视角控制**: 玩家滑动屏幕（Swipe），可以 360 度旋转中间的 3D 大方块阵列，观察内部和背面的方块分布。
- **操作方式**: 点击选中（Tap）。选中蓝块，再点击目标方块执行合成或吞噬。
- **特色功能 - 剖面切割 (横纵截面快捷选择)**:
  - 界面提供层级快捷按钮，点击后透明化其余层，解决 3D 遮挡痛点，方便精准操作。

## 三、 游戏心流与进阶系统 (空间扩容机制)

游戏彻底摒弃传统的线性闯关，采用**“空间扩容解锁”**的纯无尽机制。阵列越大，容错率越高，越能支持长线运营与高分冲榜。

### 1. 2x2x2 微缩战场 (新手与破冰)
- **定位**: 默认解锁，兼顾新手教学与初期体验。
- **体验**: 空间极度逼仄（仅8个格子），极易 Game Over。玩家在此快速熟悉规则。
- **解锁条件**: 历史最高分达到 **3,000分**，解锁 3x3x3 战场。

### 2. 3x3x3 标准战场 (核心体验区)
- **定位**: 经典尺寸，策略性与爽感开始展现。
- **体验**: 拥有 27 个格子，玩家可以开始尝试复杂的排兵布阵和连击（Combo）。
- **解锁条件**: 历史最高分达到 **20,000分**，解锁 4x4x4 战场。

### 3. 4x4x4 高维战场 (高手进阶区)
- **定位**: 高难度的脑力博弈。
- **体验**: 64 个格子，中心区域的视线遮挡成为主要难点，重度依赖“剖面切割”功能，分数产出极高。
- **解锁条件**: 历史最高分达到 **100,000分**，解锁 5x5x5 战场。

### 4. 5x5x5 极限神域 (最终冲榜场)
- **定位**: 终极挑战与全球榜单的“神仙打架”区。
- **体验**: 125 个格子，是首发版唯一的终局冲榜舞台，目标单局时长控制在 4~8 分钟。

## 四、 局内道具系统 (解场与保分)
首发版移除金币体系。道具不通过广告购买，仅通过吃掉**金黄色方块**产出，用于打破僵局与延长高分对局。
1. **精准打击 (小炸弹)**: 选中并摧毁任意一个方块（解红块卡位）。
2. **空间降维 (降级锤)**: 使任意一个红块等级 -1。

## 五、 商业化与经济系统 (极简广告变现)
游戏采用 **纯 IAA (激励视频广告)** 变现，无科技树、无天赋升级，强调公平竞技。

- **积分 (Score)**: 游戏唯一核心数值，用于排行榜竞技和解锁更高维度战场。

### 激励视频核心坑位
1. **局内复活 (限1次)**: 棋盘满时，看广告清空关键阻塞红块保分。
2. **原则**: 广告仅用于“续命”，**不直接影响排行榜分数结算**。局内道具完全由玩法产出。

## 六、 社交与长线留存 (排行榜系统)
- **【无尽之王】全球总积分榜**: 不分战场维度，纯看最终单局得分（高维战场天然具有高分优势，驱动玩家解锁高维）。每周日重置。
- **高亮炫耀**: 榜单强制展示玩家的**最终积分**与**本局最高合成等级徽章**，强化竞技炫耀与“再来一局”的冲榜欲望。
