# CubeFight - 美术与皮肤设计 (Skin Design)

**作者**: 樊老师 (主策划)
**更新日期**: 2026-03-14

## 一、 核心视觉表现定调
基于《CubeFight》的 3D 解谜合成玩法，美术风格走 **高级极简风 (Premium Minimalist)**，拒绝廉价感。

1. **外观统一**: 所有皮肤基础的 Collision Box 始终为 `1 x 1 x 1`，保证 3D 密集阵列排列整齐不穿模。
2. **材质致胜**: 采用磨砂玻璃、半透明亚克力、霓虹自发光等材质，突出内部结构。
3. **色彩克制**: 严格遵循 红(怪)、黄(财宝)、蓝(玩家) 的三原色逻辑，确保玩家一眼辨识。

## 二、 1-9 级视觉成长体系 (内部空间复杂度)
不靠“变大”，靠“变复杂”。
- **Lv 1**: 半透明空心外壳。
- **Lv 3**: 内部出现发光核心（小球体）。
- **Lv 6**: 核心演变为复杂几何体并带旋转动画。
- **Lv 9 (神级)**: 核心爆发出刺眼光芒，甚至带有微缩星系的粒子特效。吞噬时全屏冲击波。

## 三、 皮肤阶层与商业化定价 (Skins)
这是玩家消耗局内“金币”和看广告的唯一长线追求。

| 皮肤系列 | 定位 | 特色 | 获取途径 / 价格 |
| :--- | :--- | :--- | :--- |
| **【原味冰块】** | 默认皮肤 | 磨砂半透明质感，经典耐看 | 初始免费自带 |
| **【赛博霓虹】** | 进阶皮肤 | 纯黑虚空方块，边缘霓虹灯带勾勒，科技感十足 | 金币购买 (如: 10,000 Coin) |
| **【异星晶体】** | 史诗皮肤 | 突破正方体，变为不规则矿石切割，折射极强 | 金币购买 (如: 50,000 Coin) |
| **【盲盒公仔】** | 终极皮肤 | 透明外壳内含 3D 微缩角色（蓝=骑士, 红=恶龙, 黄=金币堆） | **专属广告解锁 (如: 累计观看 20 次 RV)** |

## 四、 皮肤对局内爽感的附加影响 (非数值)
更换皮肤不仅是换模型，必须提供情绪价值：
- **【专属音效】**: 机甲皮肤合成时发出金属齿轮声；晶体皮肤发出清脆玻璃碎裂声。
- **【特殊特效】**: 赛博皮肤吞噬红块时，红块不是碎裂，而是化为“数字乱码”消散；神话皮肤吞噬时，蓝块微缩骑士拔剑斩击。
