# CubeFight - 核心系统设计 (System Design)

**作者**: 樊老师 (主策划)
**更新日期**: 2026-03-14

## 一、 双轨经济模型与变现

为了保护长线皮肤生态，游戏将“表现荣誉”与“购买力资产”严格分离。

1. **积分 (Score)**
   - **定位**: 纯荣誉数值，展示在单局结算与无尽排行榜。
   - **产出**: 局内所有合成/吞噬操作均产出积分（指数级增长）。
   - **消耗**: 无。

2. **金币 (Coin)**
   - **定位**: 唯一的购买力货币，用于购买 3D 方块皮肤。
   - **产出**: 
     - **核心来源**: 局内蓝块吃掉“黄块（财宝）”掉落（等级越高，掉落呈指数级增加）。
     - **辅助来源**: 闯关模式首次通关固定奖励。
     - **倍增来源**: 结算界面观看激励视频（x2 或转盘 x2~x5）。
   - **消耗**: 皮肤商店。

## 二、 核心广告体系 (IAA / 激励视频)

在 CrazyGames 平台上，IAA 是主要变现手段。通过“限制次数”保证游戏性，通过“利益最大化”吸引点击。

| 广告坑位 | 触发场景 | 玩家收益 (强驱动力) | 每日/单局限次 | 预期转化率 |
| :--- | :--- | :--- | :---: | :---: |
| **局内复活 (Save Me)** | 棋盘填满 Game Over 时弹窗 | 清空当前最高级蓝块周围的所有红块，继续冲刺高分 | 1 次/局 | 极高 (为了保分) |
| **限时道具 (Help!)** | 右下角道具栏为空，点击 [+] 号 | 立即获得指定道具（小炸弹 / 降级锤）解场 | 2-3 次/局 | 高 (为了保命) |
| **结算翻倍 (Double/Spin)** | 胜利/失败结算面板 | 本局获得的所有金币收益翻倍或转盘抽大奖 | 无限次 | 极高 (核心收益) |
| **皮肤解锁 (Exclusive Skin)** | 皮肤商店的“神话级皮肤” | 累计观看进度条 (如 1/20)，满后永久解锁特效皮肤 | 无限次 | 高 (长线追求) |
| **插屏广告 (Interstitial)** | 关卡模式中，每通过 3~5 关的加载间隙 | （强制展示，无直接收益）作为平台基础保底收入 | 随关卡进度 | 100% |

## 三、 排行榜系统 (Leaderboard)

长线留存与炫耀的核心驱动。

1. **【无尽之王】单局极限高分榜**
   - **机制**: 记录玩家在无尽模式打出的单局历史最高积分。
   - **周期**: 每周日 24:00 重置（Weekly Reset），保持新人活跃度。
   - **展示**: 排名、ID、本局积分、**本局最高合成等级 (如 Lv.9 徽章)**、**当前装备皮肤缩略图**。

2. **【方块大亨】历史总金币榜**
   - **机制**: 记录玩家建号以来赚取的总金币量（即使花掉买皮肤也不扣除）。
   - **周期**: 永久累计榜。保障“肝帝”玩家的成就感。
