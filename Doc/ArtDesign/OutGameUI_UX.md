# CubeFight - 局外系统与商业化 UI 规范 (Out-Game UI/UX)

**负责人**: 国盛 (主美术)
**状态**: 已补充 (2026-03-15)
**前置参考**: `InGameUI_UX.md`, `SystemDesign.md`, `VisualTarget.md`
**核心适配原则**: **响应式 3D 运镜 (Responsive 3D Camera)** —— 完美兼容 PC (横屏/宽屏) 与 手机 (竖屏)。

---

## 1. 主界面 (Main Menu)
主界面绝不是一张死板的 2D 背景图，而是**游戏真实 3D 场景的“微距特写摄影 (Macro Shot)”**。进入游戏时没有 Loading 截断，直接通过运镜（Camera Lerp）飞入局内视角。

- **跨端布局 (Responsive Layout)**:
  - **PC 端 (横屏)**: 采用黄金比例分割。屏幕左侧 (38%) 放置大标题与菜单项（垂直排列，左对齐），屏幕右侧 (62%) 展示缓慢自转的高清 3D 盲盒方块（带强烈景深和泛光）。
  - **手机端 (竖屏)**: 采用居中堆叠。顶部 1/3 是大标题，中部 1/3 是极具张力的 3D 方块特写，底部 1/3 是大面积留白的毛玻璃主菜单（Play, Shop, Rank）。
- **视觉风格**: 延续高级极简风。背景纯色无缝，标题使用极具几何切割感的无衬线特大号英文字体（如 *Clash Display*），按钮带有柔和的毛玻璃 (Glassmorphism) 和悬浮阴影。

## 2. 关卡选择 (Level Select)
放弃廉价的“2D 九宫格带星星”网格布局。

- **交互形式**: 采用**垂直滚动 (手机) / 水平长卷 (PC)** 的“时间轴/里程碑”设计。
- **视觉展现**: 每一个关卡节点，不是一个 UI 图标，而是一个微缩的 **3D 几何核心模型**。
  - 未解锁：纯白石膏材质，覆盖浓重的阴影。
  - 已解锁：亮起专属颜色的环境光 (AO) 和辉光 (Bloom)。
  - 选中状态：镜头微微拉近，展现内部精致的 AI 贴图结构。UI 面板（毛玻璃材质）从侧面或底部丝滑滑入，显示过关条件。

## 3. 皮肤商店 (Skin Shop & 核心变现)
商店必须做成高级的 **“3D 潮玩画廊展厅 (3D Boutique Gallery)”**，这是刺激玩家消费的核心场景。

- **展示形式 (Carousel)**: 
  - 屏幕中央永远只有一个处于**最佳打光位置 (Spotlight)** 的方块模型。
  - 玩家左右滑动/点击箭头切换时，模型带有极其 Q 弹的 3D 翻转缓动（Bounce Rotation），配合清脆的机械切换音效。
  - 背景被极强的景深 (DoF) 彻底虚化，只留下绚丽的光斑 (Bokeh)。
- **CrazyGames 激励视频变现包装 (神话盲盒专属)**:
  - 坚决抛弃俗气的“看视频+1”按钮！我们要把它包装成**“能量注入 (Energy Injection)”**。
  - **视觉设计**: 神话皮肤处于一个暗淡的透明玻璃罩内。下方有一个带有霓虹赛博质感的进度条管线（如 `1/20`）。
  - **广告按钮**: 按钮文案为“注入能量 (Inject Energy)”，旁边带有一个极简的“播放键”小图标。
  - **看广告反馈**: 玩家看完广告返回，触发极其酷炫的管线充能特效（流光扫过，玻璃罩亮度提升 5%）。当进度满时，玻璃罩碎裂，爆发出全屏的高质量 3D 粒子和色散特效 (Chromatic Aberration)，揭晓内部极其精美的潮玩小人。高级感与变现效率完美结合。

## 4. 排行榜 (Leaderboard)
强调“阶级感”与“荣誉炫耀”。

- **排版结构**:
  - 全屏半透明磨砂黑/白底板，透出背后的 3D 场景。
  - 列表采用极其干净的横向条目设计，字号对比拉开。
- **专属炫耀点**:
  - 排名前三的玩家，其 ID 旁边不仅有流光徽章，还要**直接渲染出他们当前佩戴的 3D 皮肤缩略图**（在 UI 上开个小视口或者用高质量 RenderTarget）。这能极大刺激其他玩家去商店获取同款皮肤。
  - 玩家个人的排名永远以一块悬浮的、带高光的特殊高亮面板固定在屏幕最底部（手机）或最右侧（PC）。
