# CubeFight - 局外系统与商业化 UI 规范 (Out-Game UI/UX)

**负责人**: 国盛 (主美术)
**状态**: 已补充 (2026-03-15)
**前置参考**: `InGameUI_UX.md`, `SystemDesign.md`, `VisualTarget.md`
**核心适配原则**: **响应式居中排布 (Centralized Responsive Layout)** —— 完美兼容 PC 与 手机。

---

## 1. 主界面 (Main Menu)
为了保证 PC 与手机端布局的一致性，放弃高开发成本的“视差双栏”与“无缝运镜”，采用**绝对居中的轴对称美学 (Symmetrical Aesthetics)**。

- **视觉中心与 UI 布局 (Header Branding & Centered Action)**:
  - 放弃传统的“上Logo-中模型-下按钮”拥挤布局，改为更具人机工学与品牌感的 **“顶部图文结合，中部留给交互”** 的结构。
  - **顶部品牌区 (Top Header - Logo & Hero Prop)**: 
    - 巨大的游戏 LOGO (极简几何无衬线字体) 与 **3D 核心方块特写 (Hero Prop)** 进行组合，形成统一的“品牌商标 (Brand Mark)”。
    - **排版与缩放规范 (Icon Scale & Alignment)**: 3D 方块作为 Logo 的专属 3D Icon 放置在文字左侧。**严禁方块过大！** 方块的视觉边界高度（Bounding Box Height）必须缩小至与 `CUBEFIGHT` 文字的字高（Cap-height）**平齐或略大 1.2 倍**。方块与首字母 'C' 之间需留出合适的安全边距，确保整体看起来是一个精致紧凑的横向组合 `[ 3D Icon ] CUBEFIGHT`。
    - **Logo 动态变色特效 (Dynamic Gradient Logo)**: 为了呼应游戏内“蓝(玩家)、红(敌人)、黄(金币)”的三原色核心玩法，Logo 文字 `CUBEFIGHT` 需叠加流动的 CSS 线性渐变 (Linear Gradient)。颜色在蓝、黄、红之间进行平滑过渡。
  - **核心方块特效表现 (Hero Prop VFX)**: 缩小后的 3D 专属 Icon 依然保留极高的特效规格，使其成为整个屏幕的点睛之笔：
    - **同步变色呼吸核 (Sync-Color Breathing Core)**: 外壳内部的自发光核心 (Emissive Core) 必须与右侧文字 Logo 的变色频率保持一致！随着时间的推移，核心光团在 **冰川蓝 -> 银杏黄 -> 珊瑚红** 之间进行极其丝滑的 Lerp 颜色渐变，并伴随正弦波 (`Math.sin`) 的明暗呼吸发光，仿佛是驱动整个游戏的能量源。
    - **多轴错位嵌套线框 (Nested Axis Wireframes)**: 透明外壳内部包含数个相互嵌套、在不同轴向以极慢速度异步旋转的发光线框。
    - **鼠标/重力感应视差 (Gyro Parallax)**: 玩家移动鼠标或倾斜手机时，方块发生微小的透视旋转跟随，并在边缘产生锐利的白色菲涅尔高光 (Fresnel Glossy Highlight)。
  - **中部交互区 (Center Action Area)**: 
    - 将核心按钮组 (Play, Shop, Leaderboard, Settings) 整体上移至屏幕正中央（视觉黄金分割点附近）。
    - 优势：手机端拇指操作更加舒适（无需费力去够底部的按钮），且完美的毛玻璃按钮不会再遮挡 3D 核心方块的细节。
  - 背景为带有强烈景深 (DoF) 和渐变光影的微缩场景环境（纯色虚化底色），底部大面积留白以增加呼吸感。
- **转场过渡 (Transitions)**:
  - 点击 Play 后，采用极其干净利落的 **纯色遮罩滑移 (Solid Color Wipe)** 或 **全屏毛玻璃模糊加深 (Blur Fade)** 过渡到局内。不追求复杂的 3D 摄像机飞梭，确保加载稳定和代码极简。

## 2. 关卡选择 (Level Select)
放弃廉价的“2D 九宫格带星星”网格布局。

- **交互形式**: 采用**垂直滚动 (手机) / 水平长卷 (PC)** 的“时间轴/里程碑”设计。
- **视觉展现**: 每一个关卡节点，不是一个 UI 图标，而是一个微缩的 **3D 几何核心模型**。
  - 未解锁：纯白石膏材质，覆盖浓重的阴影。
  - 已解锁：亮起专属颜色的环境光 (AO) 和辉光 (Bloom)。
  - 选中状态：镜头微微拉近，展现内部精致的 AI 贴图结构。UI 面板（毛玻璃材质）从侧面或底部丝滑滑入，显示过关条件。

## 3. 皮肤商店 (Skin Shop & 核心变现)
商店必须做成高级的 **“3D 潮玩画廊展厅 (3D Boutique Gallery)”**，这是刺激玩家消费的核心场景。

- **展示形式 (Carousel)**: 
  - 屏幕中央永远只有一个处于**最佳打光位置 (Spotlight)** 的方块模型。
  - 玩家左右滑动/点击箭头切换时，模型带有极其 Q 弹的 3D 翻转缓动（Bounce Rotation），配合清脆的机械切换音效。
  - 背景被极强的景深 (DoF) 彻底虚化，只留下绚丽的光斑 (Bokeh)。
- **CrazyGames 激励视频变现包装 (神话盲盒专属)**:
  - 坚决抛弃俗气的“看视频+1”按钮！我们要把它包装成**“能量注入 (Energy Injection)”**。
  - **视觉设计**: 神话皮肤处于一个暗淡的透明玻璃罩内。下方有一个带有霓虹赛博质感的进度条管线（如 `1/20`）。
  - **广告按钮**: 按钮文案为“注入能量 (Inject Energy)”，旁边带有一个极简的“播放键”小图标。
  - **看广告反馈**: 玩家看完广告返回，触发极其酷炫的管线充能特效（流光扫过，玻璃罩亮度提升 5%）。当进度满时，玻璃罩碎裂，爆发出全屏的高质量 3D 粒子和色散特效 (Chromatic Aberration)，揭晓内部极其精美的潮玩小人。高级感与变现效率完美结合。

## 4. 排行榜 (Leaderboard)
强调“阶级感”与“荣誉炫耀”。

- **排版结构**:
  - 全屏半透明磨砂黑/白底板，透出背后的 3D 场景。
  - 列表采用极其干净的横向条目设计，字号对比拉开。
- **专属炫耀点**:
  - 排名前三的玩家，其 ID 旁边不仅有流光徽章，还要**直接渲染出他们当前佩戴的 3D 皮肤缩略图**（在 UI 上开个小视口或者用高质量 RenderTarget）。这能极大刺激其他玩家去商店获取同款皮肤。
  - 玩家个人的排名永远以一块悬浮的、带高光的特殊高亮面板固定在屏幕最底部（手机）或最右侧（PC）。
