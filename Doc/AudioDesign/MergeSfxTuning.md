# Merge SFX Tuning

更新时间：2026-03-25

## 目标

- 目标音效：`merge_lv2` ~ `merge_lv9`
- 目标听感：保留升级、合体、往上拱的爽感，但把“硬亮刺耳”的部分磨圆
- 设计原则：不能把合并做得没劲，要是“脆亮升级”而不是“尖锐报警”

## 调整策略

### 频段
- 主体从偏硬的 `square` 改为更柔一点的 `triangle`
- 保留上行琶音骨架，维持升级反馈和节奏推动
- 高频只保留极轻一层泛音辅助，避免连续合并时上中高频堆积

### 包络
- 每级音符保持短促，但攻击更顺滑
- sustain 略收短，减少长时间占用频段
- release 略加长一点点，让结尾不至于像针扎一样断掉

### 混音 / 播放
- 运行时播放增益下调：`1.1 -> 0.94`
- 仍保留 ducking，让合并音效在 BGM 上方有存在感
- 同类实例上限保持 `2`，避免连续连锁时爆亮、爆音

## 交付备注

- 本次仅柔化 timbre 与增益，不修改 cue 名称与时间切片规则
- 若后续仍觉得亮，可继续优先下修高频辅助层，而不是直接砍掉主琶音
